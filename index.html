Three Variables In JavaScript

  • var
  • let
  • const

5 Legal Variable Names

  • Letters(both uppercase or lowercase)

  • Numbers

  • Underscores

  • $ Dollar Sign

5 Illegal Variable Names

  • A variable name cannot start with a numeral

  • You cannot have a mathematical or logical operator in a variable name.

  • You must not use any punctuation marks of any kind in a JavaScript variable name, other than the underscore.

  • JavaScript names must not contain spaces.

  • You cannot use JavaScript keywords for variable names.
